# 仮想通貨通知システム

[Download here](https://github.com/smokker-oldman/crypto_robo-2t/releases)

## 機能

- 暗号通貨のレート情報表示
- 価格アラート
- 自動取引

## 使用技術

- **フロントエンド**

  - **React**
  - **TypeScript**

- **バックエンド**
  - **Node.js**
  - **TypeScript**

## 開始手順

### 1. リポジトリをクローン

git clone https://github.com/aiba-22/crypto_robo.git

### 2. Docker コンテナのビルドと起動

docker-compose up
フロントエンド、バックエンド、バッチ、データベースが立ち上がります。

### 3. アプリケーションへのアクセス

http://localhost:8000
